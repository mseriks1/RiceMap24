# Marketing Coach – Month 01-A  
## Focus: Launch your social media system — the right account structure + first setup on each platform (ready to market)

This is the very first Marketing Coach. The goal is not to post a lot. The goal is to **set up social media correctly** so you get a system that:

- looks safe and professional to new customers  
- is easy to run without stress  
- makes ordering simple (one clear link)  
- lets you use both “human” (trust) and “brand” (professionalism)

In other words, you’re building the **platform foundation**.

Early key term:  
CTA = Call To Action (what you want the customer to do right now). Use one CTA per post/text/bio:  
**“Menu + ordering: [link]”**  
*(This becomes your standard CTA across all of RiceMap24.)*

⸻

## First: Private vs business — what’s smartest?
This matters, because many people make this messy.

### Private account (personal profile)
**Use private when you want to build trust and human warmth:**
- “Hi, I’m cooking at home today”  
- behind-the-scenes  
- small everyday updates  
- you as a person (smile, short explanation)

**Pros**
- often better organic reach  
- feels real and close  
- easier to get local support (friends/neighbors)

**Cons**
- can mix private life and business  
- less “professional first impression” for brand-new customers

### Business account (brand/profile for the kitchen)
**Use business when you want to be clear, professional, and easy to order from:**
- menu + ordering link  
- pinned post (“Start here”)  
- highlights  
- clear identity + brand name  
- reusable content

**Pros**
- looks professional and safe  
- becomes an asset you can grow long-term  
- easier to keep one clear link and structure  
- easier to scale (you can add helpers later)

**Cons**
- often slower organic reach at the beginning  
- can feel “less personal” if you never show the human side

### The smartest structure for home kitchens (recommended)
✅ **Use BOTH.**  
This is the most effective structure for most RiceMap24 kitchens:

1) **Business profile/page** = the “storefront”  
2) **Private profile** = the “trust channel” (human presence)

**Rule:**  
- The business profile is where the customer should *end up* to order (the clean structure).  
- The private profile is how you create *local familiarity* and drive people in.

⸻

# Brand-kit (must be created before you set everything up)

## Brand-kit (5 things)
Before you create accounts everywhere, make these 5 things:

1) **Kitchen name** (consistent everywhere)  
2) **Profile image** (logo OR a strong food photo)  
3) **Cover/hero image** (wide image for FB + can be used as a hero elsewhere)  
4) **Short bio (2–3 lines)**  
5) **One ordering link + standard CTA line**

### Recommended naming style
Keep it simple and searchable:
- Kitchen name + area/city  
  Example: “Maria’s Filipino Kitchen – Fredrikstad”
- Or: kitchen name + cuisine  
  Example: “Maria’s Filipino Kitchen”

### Short bio template (copy/paste)
Small-batch home kitchen in [area]. Authentic [cuisine].  
Menu + ordering: [link]  
Pickup in [area]. Allergens listed in menu text.

### Standard CTA (use everywhere)
**Menu + ordering: [link]**

# Platform-by-platform: creation + the first things you should do

## Facebook
### What you create
✅ **A Facebook Page (business page) for the kitchen**

### First setup (checklist)
1) Create page name + category (Food/Restaurant/Kitchen)  
2) Add profile image + cover image  
3) Fill “About” (short bio) + area + contact info  
4) Add ordering link (RiceMap24 link)  
5) Add opening/pickup info (simple)  
6) Add a “Start here” pinned post (see below)

### First action (important)
✅ Make a pinned post that tells customers exactly what to do.

**Pinned post template (copy/paste):**
**Start here 👇**  
I run a small home kitchen in [area] and cook [cuisine/style] in small batches.  
**Menu + ordering:** [link]  
Pickup in [area]. Allergens listed in the menu text.

### Smart use of your private profile (without spamming)
Use your personal profile to:
- share the pinned post once  
- share weekly menu updates (lightly)  
- comment and be present in local groups (as a person)

**Rule:** Don’t post the same sales message every day.  
Be human, local, and helpful.

⸻

## Instagram
### What you create
✅ **Instagram Business Account** (or Creator — both are fine, Business is simplest)

### First setup (checklist)
1) Switch to Business/Creator  
2) Set the same name + handle (as close as possible to your other platforms)  
3) Add profile image  
4) Add short bio (2–3 lines)  
5) Add the ordering link (RiceMap24)  
6) Create 3 Highlights (empty is okay at first):
   - “Menu”  
   - “How to order”  
   - “Pickup”  
7) Create a pinned “Start here” post (see below)

### First action (important)
✅ Create one post (can be a simple image) and pin it.

**Pinned post template (copy/paste):**
**Start here 👇**  
Small-batch home kitchen in [area]. Authentic [cuisine].  
**Menu + ordering:** [link]  
Pickup in [area]. Allergens listed in menu text.

⸻

## TikTok
### What you create
✅ A TikTok account for the kitchen (brand account, but human tone)

2) Bio: cuisine + area + “Menu + ordering on RiceMap24”  
3) Add link in bio (if available)
