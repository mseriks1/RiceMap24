2. Layout and collage generator

Examples: all images equal size, 1 large + small images, 2 large + 2 small, top hero + gallery, and more.

2b. Custom split draw

Build your own collage rooms by drawing split lines inside a rectangle. Each split cuts one room into two. This is a first version with straight vertical or horizontal cuts.
How to use: choose Custom Split Draw, turn on draw mode, then drag inside one room. Mostly sideways = vertical split. Mostly up/down = horizontal split.

2c. Snap draw mosaic

Draw a free split line inside the rectangle. The start and end points snap to edges, corners, and existing split points so you can build more organic collage rooms.
How to use: choose Snap Draw Mosaic, turn on snap draw mode, then drag across one room. The line snaps to that room’s edges or vertices and cuts it into two new rooms.
